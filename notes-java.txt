Git

Git is a version control system used to track changes in code and manage different versions of a project.

GitHub

GitHub is an online platform that stores Git repositories and helps developers collaborate, share, and manage code.

Git Bash

Git Bash is a command-line tool that lets you use Git commands on Windows using a Linux-like terminal.

Command	Use ------
git init	        Create a new Git repository
git clone <url>	        Copy a repository from GitHub
git status	        Check file status
git add .	        Add all files for commit
git commit -m "msg"	Save changes with message
git push	        Upload code to GitHub
git pull	        Download latest changes
git branch	        Show branches
git checkout            <branch>	Switch branch
git merge <branch>	Merge branches

HOW TO CREATE A BRANCH -

git branch branch_name
→ Creates a new branch

git checkout branch_name
→ Switches to the branch

git checkout -b branch_name
→ Creates and switches to a new branch

git branch
→ Shows all branches

git branch -d branch_name
→ Deletes a branch
